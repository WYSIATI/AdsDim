# AdsDim

**广告变暗,好帖发光 —— 从不隐藏任何帖子。**

[English](./README.md) | 简体中文

AdsDim 是一个开源的 MV3 Chrome 扩展,在 x.com 时间线上**标记**广告(硬广、软广、疑似),但**绝不隐藏或删除任何帖子**。你看到的信息流永远是完整的 —— AdsDim 只是让真实内容更亮眼、让广告安静下来。

## 特性

- 🏷️ **只标记,不隐藏** —— 每条帖子都还在,广告只是被打上 `硬广 / 软广 / 疑似` 徽章并淡出。
- 🔍 **本地检测** —— 官方推广标识(支持 12 种界面语言:Ad / Promoted / 广告 / 推广 / プロモーション / Anzeige / Publicidad / Publicité……)+ 软广启发式(#ad、优惠码、返利链接、加微信等模式)。
- 🎨 **三种视觉方案 × 两档对比度** —— 默认 **玻璃聚焦 + 强烈**:
  - **玻璃聚焦 Glass Focus** —— 真实内容浮起为磨砂玻璃卡片,广告降饱和退入背景。
  - **光晕通道 Glow Lane** —— 真实内容获得渐变光环与左侧光晕,广告转为灰调。
  - **剧场模式 Theater** —— 广告如剧场熄灯般变暗,真实内容保持全亮。
- 🖱️ **悬停即恢复** —— 鼠标移到被弱化的广告上,立即恢复完全可见(纯 CSS)。
- 🌍 **多语言** —— 界面已本地化(English、简体中文,欢迎贡献更多),广告标识检测不受你的 X 显示语言影响。
- 🌗 **自动深浅色主题**、零布局位移、兼容 X 虚拟滚动。

## 隐私

**默认零网络请求。** 所有检测都在你的浏览器本地完成;不采集、不上传、不追踪任何数据。设置仅存储在浏览器的扩展同步存储中。

## 开发

```bash
npm install
npm run dev        # WXT 开发模式(Chrome)
npm run build      # 生产构建
npm test           # 单元 + 集成测试(Vitest)
npm run lint       # ESLint
npm run compile    # TypeScript 类型检查
```

设计稿见 `design/`(`preview-v2-inverted.html` 是渲染器的 CSS 来源)。

### 贡献翻译

界面文案在 `src/i18n/`(每个语言一个文件,约 20 条字符串)。新增语言文件并在 `src/i18n/index.ts` 注册后提 PR 即可。代码注释与 commit 信息使用英文。
